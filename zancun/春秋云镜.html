内网渗透常见工具以及用法总结

内网渗透常见工具以及用法总结

内网信息收集

fscan扫描内网网段

PsExec.exe

目的:为了获得目标靶机的system权限

mimikatz

为了从内存中读取hash账号密码

只要域管理员用户使用自己的域控账号登陆了服务器，那么可以用mimikatz来抓取域控的账密

就算内网的机器打了KB2871997补丁，同时注册表中关闭Wdigest Auth，也可以抓取到hash值，

也可以抓取krbtgt的sid和hash值，krbtgt就是KDC的超级管理员用户

rdp远程登陆

可以在cs中注入远程连接的用户进程来实现横向移动

msf hash 传递

可以使用merterpreter中自带的hashdump功能，但是要高权限

可以在merterpreter中 run post/windows/gather/hashdump，无需高权限

得到hash值之后可以用以下三个模块来实现hash传递攻击，而且可以直接拿到meterpreter

Crackmapexec

可以对一个网段的主机进行票据攻击，kali中配proxychains直接用

wmiexec

impacket的其中一个脚本，同样记得走porxychains代理

GetNPUsers.py

Impacket 工具包中的一个 Python 脚本，用于尝试枚举域用户，通常情况下，预身份验证是默认开启的，但一旦关闭了(Do not require Pre-Authentication)，攻击者就有可能利用指定的用户名，通过向域控制器的Kerberos 88端口请求票据TGT

这种攻击也被称为Kerberos AS-REP Roasting 攻击：这是一个针对 Kerberos 认证协议的攻击方法，主要目标是尝试从域内获取不需要预身份验证的用户的加密凭证信息。这种攻击可以尝试暴力破解密码，特别是针对没有配置 Kerberos 预身份验证的用户账户。

hashcat

可以和GetNPUsers.py联用，如果得到了某一个域用户的hash值，那么就可以用hashcat去爆破出明文密码，就可以实现rdp登陆了

SharpHound.exe

BloodHound 攻击图谱工具的采集器组件，可以从域环境中收集信息

smbpasswd.py

如果密码喷洒(Crackmapexec)时提示密码过期，那么就可以用smbpasswd.py来修改密码

注册表相关操作

查看注册表中的IFEO权限，可以看是否所有的用户都可以修改注册表
